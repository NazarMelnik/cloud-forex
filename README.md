# Cloud Forex Servers

React + TypeScript + CSS — тарифні картки Cloud Forex.

## Запуск

```bash
npm install
npm run dev
```

- `buildOrderUrl` — реальний URL checkout
- `filterForex` — уточнити критерій, якщо треба `title_tag.forex_server`
